# Versitron SF70460MPA | 4-Port Managed Industrial PoE+ Switch with Dual SFP For System Integrators

The Versitron SF70460MPA is a rugged 4 port managed industrial PoE+ switch built for system integrators deploying reliable networks in demanding environments. It features two Gigabit RJ45 PoE+ ports and two SFP fiber uplinks, allowing seamless integration between powered edge devices and long-distance fiber infrastructure.
